Government research identifier service

Use this service to:

  • create a unique government research identifier service ID (GRIS ID)
  • view an existing study
  • make changes to an existing study

Before you start

To create a new GRIS ID, you'll need:

  • the short title of the study
  • an ORCID iD for the chief investigator or a key team member

It would be helpful to know the details of any related study, such as the GRIS ID, IRAS ID, and funding ID.

To view or make changes to a study, you'll need the:

  • the study's existing GRIS ID
  • the chief investigator's email address
